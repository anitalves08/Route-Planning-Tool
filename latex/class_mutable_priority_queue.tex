\subsubsection*{template$<$class T$>$\newline
class Mutable\+Priority\+Queue$<$ T $>$}
Fila de prioridade mutável baseada em heap binário. 

class T must have\+: (i) accessible field int queue\+Index; (ii) operator\texorpdfstring{$<$}{<} defined.

Esta classe fornece as operações necessárias para o algoritmo de Dijkstra, permitindo a inserção, extração do mínimo e diminuição da chave de elementos na fila de prioridade.
