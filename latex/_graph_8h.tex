\doxysubsection{Detailed Description}
Declaração das classes e funções para manipulação de grafos. 

Contém a definição de classes para Vértices e Arestas, além da classe principal \doxylink{class_graph}{Graph} que representa um grafo. As funções são utilizadas para adicionar e remover vértices e arestas, entre outras operações.
